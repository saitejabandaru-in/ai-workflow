## What Is an AI Workflow?

A sequence of AI-powered steps that transform inputs into outputs automatically:

- Raw data → AI extraction → AI transformation → AI validation → Structured output
- Email → AI summary → AI classification → AI routing → CRM update
- Document → AI parsing → AI analysis → AI report → Email send

## Workflow Patterns

- **Sequential Chain** — step1 → step2 → step3
- **Map-Reduce** — split → parallel process → combine
- **Branch and Route** — classify → route to specialist
- **Loop with Memory** — iterate until condition met
